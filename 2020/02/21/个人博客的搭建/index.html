个人博客的搭建

1. 安装

  1. 安装Git

  2. 安装Nodejs

  3. 安装Hexo

输入 hexo -v 可以看到hexo的版本

2. 准备

  1. 首先新建一个空文件夹 如blog
  2. 在 blog 目录下 右键 git bash 执行 hexo init
  3. 本地启动（预览）hexo s
  4. 新建一篇文章 hexo n "我的第一篇博客文章"
  5. 在blog目录下的 source/_posts 目录下可以看到创建的文件
  6. 打开编辑即可
  7. 编辑好之后，清理一下，执行： hexo clean
  8. 然后生成，执行： hexo g
  9. 从新启动就可以看到你创建的文章，执行： hexo s
